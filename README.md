# Validity Index Calculator

A web-based tool to calculate:
- **Content Validity Ratio (CVR)** based on Lawshe's formula.
- **Content Validity Index (CVI)**: Item-level (I-CVI) and Scale-level (S-CVI).
- **Face Validity Index (FVI)**: Item-level (I-FVI) and Scale-level (S-FVI).

## How to use
1. Download the `Validity_Data_Template.xlsx`.
2. Fill in your expert ratings:
   - **CVR**: 1 (Not necessary), 2 (Useful but not essential), 3 (Essential)
   - **CVI/FVI**: 4-point scale (1 to 4)
3. Upload the file to the calculator.
4. View results and export as PDF.
